# Prompts

This directory contains the markdown prompt assets used by the chat
orchestration layer.

## Contents

- System prompts for standalone and blended execution
- Shared prompt fragments for database rules, narration, and presentation
- User-message templates that wrap specific flows like shared-report follow-ups

## Direction

- Prompt assets should stay composable and readable
- Visualization guidance will continue to evolve as MotherDuck Dive usage expands
- Prompt changes that alter behavior materially should be captured in ADRs or docs notes
